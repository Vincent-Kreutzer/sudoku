# Sudoku Game

A web-based Sudoku (Number Place) game built with HTML, CSS and vanilla JavaScript.

## Features

- 3 difficulty levels (Easy / Normal / Hard)

- Responsive design (Desktop / Tablet / Mobile)

- Original character with animated reactions for correct, incorrect, clear, and game over states

- Sound effects for correct and incorrect inputs

- Restart the game with the same difficulty

- Return to top menu from gameplay and result screens

- Clear and Game Over result screens
